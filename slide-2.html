Grundlagen: Aufbau einer Szene

Aufbau

Szene besteht aus 3 Elementen:
  • Szene - ordnet alle Objekte im Raum an
  • Kamera - bestimmt, was man von der Szene sehen kann
  • Renderer - technischer Helfer, der alles malt

Code

var viewport = document.getElementsByClassName('three-js-scene')[0]; var scene = new THREE.Scene(); var camera = new THREE.PerspectiveCamera( 75, // field of view viewport.offsetWidth/viewport.offsetHeight, // aspect ration 0.1, // min distance 10000 // max distance ); camera.position.z = 40; camera.position.y = 20; camera.lookAt(scene.position); var renderer = new THREE.WebGLRenderer(); renderer.setSize(viewport.offsetWidth, viewport.offsetHeight); // set correct dom element size renderer.setClearColor(0x000000); viewport.appendChild(renderer.domElement);
